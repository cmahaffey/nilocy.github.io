Radicalize

Github Heroku
Radicalize is a new way to look up Chinese characters. Using a full MEAN stack, in conjunction with external text documents and RegEx usage, Radicalize allows for a simplistic look-up system based off each character's radicals, the character's smaller component parts.
Radicalize uses regular expressions to parse text documents into a Mongo database, and Express.js and AngularJS to allow for an intuitive point and click system for looking up characters within the database. Express.js and RegEx are also use to scrape an external website for information on a page due to an unfortunate lack of good, free, Chinese dictionary APIs.
Born out of the frustration inherent in trying to learn to read Chinese, this project is the actualization of a longtime dream. While currently still limited due to the available data splitting Characters in this way, and lack of good free translation APIs. Radicalize is a first step in vastly simplifying looking up unknown Chinese characters.
Technologies Used: MongoDB, Express.js, Node.js, AngularJS

Soundtrak

Github Heroku
An app made to allow ways to look up the soundtracks of your favorite movies. Consumes both the OMDB and Spotify APIs, as well as a database populated based off an IMDB text file, to allow for finding all the songs to your favorite movies. Searches limited to movies beginning with "al","fro","rush" due to heroku's database limitations
Technologies Used: Ruby on Rails, BackBone.js
